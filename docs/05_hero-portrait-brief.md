# Asset brief — Dr. Mineral hero portrait (character sheet)

**Why:** the canonical face of the universe — drives the character sheet, Book I cover candidate, and the re-skinned landing page hero. The library has action poses but no clean dignified portrait. Highest-priority single asset.

**Type:** still 3D render (Nano Banana / Gemini). Generate a small set (4 variants), pick one, key it.

## Consistency anchor (paste into every variant)
> The same squirrel character across all images: warm 3D Pixar-style render, reddish-brown fur, cream/pale belly, large expressive intelligent eyes, a small dark nose, and an oversized bushy tail curving upward behind him. He holds a single acorn. Friendly, humble, curious expression. Consistent character, same proportions every time.

## The shot
- **Pose:** seated or standing in a relaxed three-quarter view, holding the acorn in both forepaws near his chest, looking slightly off-camera with a thoughtful, kind expression. Dignified but warm — a scientist's calm curiosity, not a cartoon mascot's mug.
- **Read:** he looks like an *ordinary* squirrel with an unusual spark of intelligence. **No costume, no lab coat, no glasses, no gadgets** — canon: publicly he's just a squirrel. The intelligence is in the eyes and posture, not props.
- **Camera:** eye-level or a hair below, portrait framing, head-and-body, a little negative space above.
- **Lighting:** warm cinematic key from upper left, soft rim light separating the tail from the background, gentle falloff.
- **Background:** **flat solid neutral mid-grey (#777) or flat blue** — NOT green (green spills into reddish fur and keys dirty), NOT a checkerboard, NOT a scene. A flat contrasting color so it keys cleanly afterward.

## Negatives
no chain, no rope, no leash, no text, no logo, no watermark, no UI, no checkerboard, no busy background, no extra animals, no human objects, no clothing.

## After generation
Key out the flat background with the `ai-render-keying` skill → transparent PNG → add to the character sheet and the Canon Registry visual section. (Nano Banana won't give true transparency; that's expected — generate on flat color, key second.)
