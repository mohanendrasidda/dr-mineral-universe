# District Field Guides

Deep lore, one file per district — the level of detail that makes every website feature feel like a *real place you can walk into.* Each guide follows the same template so they're consistent to read, animate, and build pages from. Summary map lives in `../district-atlas.md`; this folder is the depth behind it. New leads/citizens cast here should be mirrored into `../characters.md`.

## Template (every guide)
1. **Essence** — one line.
2. **Who runs it** — lead (species fit to the science, with a real flaw) + key citizens.
3. **Walk in** — a sensory daily scene.
4. **What they make / do** — the work, on-stage: prototypes, failures, the joy of making.
5. **Secrets & open mysteries** — hooks for stories and exploration.
6. **→ Website feature** — how the place becomes an explorable feature.
7. **Cinematic shots** — target images for animation/site.
8. **Access** — 🟢 public / 🟡 members / 🔴 restricted.
