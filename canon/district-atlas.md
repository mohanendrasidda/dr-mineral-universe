# Canon — The District Atlas

The map of the laboratory-civilization. **Every district is a living quarter AND a future website feature** — visitors explore a real place, never a menu. Consult before inventing geography; add districts here (don't rename/remove the original 15). Derived from `../docs/06_universe-bible-v2.md`. Related: [the-squirrel-network](the-squirrel-network.md) · [daily-life-and-culture](daily-life-and-culture.md) · [technology-and-engineering](technology-and-engineering.md) · [growth-and-versions](growth-and-versions.md).

Legend — **Access:** 🟢 public (any visitor) · 🟡 members · 🔴 restricted (story/clearance). **Verb:** which mission verb it serves (discover · build · protect · preserve · pass-on · live).

## III. How the atlas drives the website
- **Homepage = the campus surface;** entrances descend to districts (see network file).
- **Every feature you build = a district you reveal.** Don't ship a "changelog page" — open the **Version History Museum.** Don't ship "docs" — enter the **Archives/Library.** Don't ship a "security status" — look into **Sentinel Hall.**
- **Access tiers** (🟢/🟡/🔴) are both lore and the site's public/member/restricted model.
- **New districts are how the lab grows** — every version opens one or more. Log additions here.
